# Logbook dr. Benny v11 (OCR Integrated)

Aplikasi Logbook Medis dengan fitur pembacaan otomatis dari foto SIMRS.

## Cara Update ke GitHub:
1. Extract file zip ini.
2. Upload semua file (`LogbookV5_fixed.html`, `sw.js`, `manifest.json`) ke repository GitHub kamu.
3. Pastikan mengganti file yang lama.
4. Buka aplikasi di HP, lakukan refresh untuk memperbarui cache.

## Fitur Baru:
- Tombol **AMBIL / UPLOAD FOTO** untuk scan otomatis Nama & No. RM.
- Integrasi Tesseract.js untuk proses OCR.
